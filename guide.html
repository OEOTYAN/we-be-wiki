                    # 下载与使用 LiteLoader

                      本插件的使用需要依赖于 LiteLoaderopen in new window ，一款是一个非官方的 Minecraft 服务端插件加载器。你需要先安装此加载器才可使用 WorldEdit-BE 。

                    # 下载并安装插件本体

                      在此之前，请确保你已经安装好 LiteLoader 。之后，前往 WorldEdit-BE 的下载页面open in new window下载最新版本的文件。 下图展示了下载页面：

                      可以看到，通常需要下载的是一个 .dll 文件，你需要将其放置于

                    └───bedrock-server-xxx (服务器文件夹)
  └───plugins
    └───WorldEdit-BE.dll

                    # 下载并安装客户端用户界面

                      当你尝试使用该插件时，会发现选框没有被显示出来，这通常是因为你没有安装客户端用户界面。在 WorldEdit-BE-CUI 的下载页面open in new window下载最新版本的 .mcpack 资源包文件并安装到游戏内，以全局资源的方式加载资源包。
